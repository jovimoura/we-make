## Como rodar

Primeiro instale as dependencias do projeto:

```bash
npm install
# ou
yarn
# ou
pnpm i
# ou
bun i
```

Segundo, rode o projeto com:

```bash
npm run dev
# ou
yarn dev
# ou
pnpm dev
# ou
bun dev
```

Abra [http://localhost:5173](http://localhost:5173) com seu browser para ver o resultado
